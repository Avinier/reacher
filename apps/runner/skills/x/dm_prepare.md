# X DM Prepare Skill

## When To Use
Use for preparing X direct messages.

## Goals
Open the target profile or DM thread, stage the draft, and stop before send.

## Page Cues
DM button availability, protected account state, and message composer.

## Research Moves
Confirm handle identity before staging a draft.

## Extraction Targets
Composer availability and profile handle.

## Action Recipes
Open profile, observe DM affordance, paste draft if composer is clear.

## Failure Cues
DMs unavailable, protected account, suspended account, login wall.

## Stop Conditions
Do not click Send.

## Output Requirements
Record prepared or failed state.
