# X Page Cues Skill

## When To Use
Use while observing x.com pages.

## Goals
Classify profile, search, message, login, protected, and suspended states.

## Page Cues
`/home` means logged-in home. User paths may be profiles unless reserved words.

## Research Moves
Capture recent post text with URLs.

## Extraction Targets
Handle, display name, bio, post URL, and date cues.

## Action Recipes
Observe navigation and primary actions first.

## Failure Cues
Protected, suspended, or login prompt.

## Stop Conditions
Stop if account cannot be accessed.

## Output Requirements
Return concise page state.
