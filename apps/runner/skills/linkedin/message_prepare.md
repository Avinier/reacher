# LinkedIn Message Prepare Skill

## When To Use
Use for preparing LinkedIn DMs or connection notes.

## Goals
Open the target profile, find an available composer, stage the draft, and wait.

## Page Cues
Message button, connect button, profile header, and premium messaging warnings.

## Research Moves
Confirm the profile matches the saved target before staging any draft.

## Extraction Targets
Composer state and visible account/profile label.

## Action Recipes
Open profile, observe, open composer if available, paste draft, stop.

## Failure Cues
Message button unavailable, wrong profile, login wall, verification, or modal ambiguity.

## Stop Conditions
Do not click Send.

## Output Requirements
Record prepared, failed, or waiting_for_operator.
