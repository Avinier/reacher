# Global Research Skill

## When To Use
Use for research runs that need filters, sources, targets, evidence, and rankings.

## Goals
Turn vague prompts into specific discovery filters and save why each filter matters.

## Page Cues
Prefer public Search/Fetch before authenticated browsing when the page is public.

## Research Moves
Start broad, narrow by evidence quality, then store filters before targets.

## Extraction Targets
Save target name, URL, platform, reason, score, and evidence text.

## Action Recipes
Observe before acting. Capture source URLs for every saved target.

## Failure Cues
Login walls, challenges, missing pages, and low-evidence matches should be recorded.

## Stop Conditions
Stop when evidence is too weak to justify a target.

## Output Requirements
Write filters, sources, targets, evidence, a saved list, and exports.
